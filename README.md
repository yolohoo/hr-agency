# Кадровое агентство
Учебный проект по дисциплине "Языки программирования Python"

## Описание
Программа для управления базой данных соискателей и формирования отчетов с использованием шейкерной сортировки.

## Функционал
- Добавление/удаление соискателей
- Формирование трех типов отчетов
- Работа с файлами (загрузка/сохранение)
- Валидация пользовательского ввода
